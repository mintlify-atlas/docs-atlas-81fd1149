## Get started in three steps

<Steps>
  <Step title="Install the SDK">
    Add the SDK to your project using npm, or load it directly via a `<script>` tag in HTML.

    ```bash
    npm install @kissflow/lowcode-client-sdk
    ```
  </Step>
  <Step title="Initialize">
    Call `KFSDK.initialize()` inside your component to get the `kf` instance. This must run inside the Kissflow platform iframe.

    ```javascript
    import KFSDK from '@kissflow/lowcode-client-sdk'

    let kf
    ;(async function () {
      kf = await KFSDK.initialize()
    })()
    ```
  </Step>
  <Step title="Start building">
    Access forms, pages, components, and more through the `kf` object.

    ```javascript
    // Read current user details
    const { Name, Email, AppRoles } = kf.user

    // Get a form field value
    const value = await kf.context.getField('fieldId')

    // Show a toast notification
    kf.client.showInfo('Hello from your custom component!')
    ```
  </Step>
</Steps>
